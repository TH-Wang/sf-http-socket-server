# sf-http-socket-server

### 环境

先安装依赖的包

```git
$ npm install
```

或者

```git
$ yarn
```

### 运行服务

```git
$ npm run start
```

或者

```git
$ yarn start
```

### api文档

需自行下载 `apidoc`

```git
$ npm install apidoc -g
```

或者
```git
$ yarn global add apidoc
```

编写完apijs文件后，生成文档

```
$ npm run apidoc
```

或者

```
$ yarn apidoc
```
